# Frontend Autopilot Example

This bundle turns `ui/options-site.html` into a Vite-built artifact served by
Tinkabot at:

```text
/artifacts/bundle/frontend-autopilot/index.html
```

The trusted shell loads that artifact as generated content and mediates its
`item_submit` command. The generated UI never receives raw substrate authority.

The checked-in HTML is a bootstrap placeholder. The frontend-autopilot proof
replaces it in the release package with bytes generated by Claude Opus before
Tinkabot starts.
