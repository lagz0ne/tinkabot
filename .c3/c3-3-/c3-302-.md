## Goal

Bridge durable items to local action.

## Parent Fit

| Field | Value |
| --- | --- |
| Parent | c3-3 Tinkalet Edge |
| Role | Item records, app-action ingress, watches, cursors, server-owned schedules, and local reaction execution |

| Does not own | Bundle manifest filter graph or runtime-owned NATS accounts |

## Purpose

This component implements the edge side of chain reaction outside bundles: Tinkabot records durable item truth, Tinkalet submits app actions or watches/waits through a profile, and an explicit local reaction can run a local command and write back a product item only when profile authority permits it.
